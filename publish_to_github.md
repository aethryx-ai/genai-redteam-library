# Publish to GitHub

Create a repository named `aethryx-genai-redteam-library`, then run:

```bash
git init
git add .
git commit -m "Initial Aethryx GenAI red team scenario library"
git branch -M main
git remote add origin https://github.com/aethryx-ai/aethryx-genai-redteam-library.git
git push -u origin main
```
